Groupie-Tracker

Description du projet

Groupie Tracker est un site codé en html et en css qui, grâce à une API donnée, permet de rechercher et d'afficher des informations concernant les artistes présents sur la plate-forme (à noter que le projet avait initialement été réalisé grâce à l'API de Spotify, mais due aux contraires du projet cela vu être modifié).
Ce projet a été réalisé en groupe avec Pablo LANCEL, Youmna LARABI et Gloria Stella KOUALIAGNIGNI NJUNDOM.

Objectif du projet

L'objectif était de réaliser un site codé en html et css qui permet d'afficher des informations sur des artistes dans une base de données (l'API utilisée était donnée et à exploiter à l'aide d'un code en Golang). Une barre de recherche est présente sur la page et permet de rechercher les artistes en fonctionne d'une chaîne de caractère : si celle entrée dans la barre de recherche est présente dans le nom d'un des artistes, alors celui-ci s'affichera. Une fois qu'on a trouvé l'artiste recherché, il est possible d'accéder à des informations sur lui comme par exemple son nom (ou le nom des membres du groupes), la date du début de carrière, etc...
